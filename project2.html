Overview

In The Elemental Carnival, players travel around different timelines as mystical elves in order to save the world against the evil forces. The game can be played with up to 4 people, with each player controling one of four special elves with different elemental powers. Players must each harness their own distinct abilities and work together to solve puzzles and defeat enemies.

As the game progresses, a shop will be available to players where they can buy items such as speed potions, and unlock element combinations. Elves can gain access to much stronger abilities temporarily when two elements are combined.

Key Features

  • Local Coop multiplayer for up to 4 players
  • 4 base elements i.e Fire, Water, Wind, Earth. With element combinations, results in a total of 10 elements. Distinct VFX for each element, designed via the Shuriken Particles System
  • 4 original characters, created from scratch using Blender
  • Interactive Shop UI
  • Intelligent enemy AI trained using Reinforcement Learning powered by Unity Machine Learning Agent Toolkit
  • Save & Load functions for level and puzzle progress
  • Dynamic hair bounce via hair bone rigging and joints
